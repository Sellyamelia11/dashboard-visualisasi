# Dashboard ini menampilkan data atlet disabilitas DKI Jakarta dalam bentuk visualisasi interaktif menggunakan Streamlit, Pandas, dan Plotly.

# Instalasi Project

1. Clone Repository
   # https://github.com/Sellyamelia11/dashboard-visualisasi.git
   # cd folder-project

2. Instalasi Python & Virtual Environment
   # python -m venv venv
   # masuk ke > venv\Scripts\activate

3. Install Dependencies
   # pip install streamlit pandas plotly openpyxl

4. Jalankan
   # streamlit run dashboard_atlet.py
